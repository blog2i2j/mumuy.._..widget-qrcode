# 基于Web component的二维码组件

## 演示地址
[https://passer-by.com/widget-qrcode/](https://passer-by.com/widget-qrcode/)


## 组件安装
```javascript
npm run widget-qrcode
```


## 组件使用
```html
<widget-qrcode value="https://passer-by.com/"></widget-qrcode>
```
